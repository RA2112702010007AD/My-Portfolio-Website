# ⚡ Getting Started

Clone the repository

```bash
git clone https://github.com/RA2112702010007AD/My-Portfolio-Website.git
```

Navigate to the project

```bash
cd My-Portfolio-Website
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Open your browser

```
http://localhost:3000
```
